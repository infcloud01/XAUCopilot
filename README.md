## üöÄ Overview

**XAUCopilot** is an autonomous multi-agent AI system designed to simulate a professional trading desk. Unlike simple trading bots that rely on a single script, XAUCopilot utilizes **CrewAI** to orchestrate two distinct personalities that debate and collaborate to generate high-probability trade signals for Gold (XAU/USD).

*   **Agent 1: The Research Manager** (Fundamental Analysis) - Scours the web for the latest Fed news, inflation data, and geopolitical events to determine the macro sentiment.
*   **Agent 2: The Technical Analyst** (Quant/Technical Analysis) - Reads price data, calculates indicators (RSI, EMA), and combines the Manager's sentiment with chart logic to execute trades.

## ‚ú® Key Features

*   **100% Free Data Tools:** Uses `ddgs` (DuckDuckGo) for news and `yfinance` (Yahoo Finance) for price data. No expensive API subscriptions required.
*   **Custom Math Engine:** Includes a custom Python Tool that calculates **RSI (14)** and **50 EMA** locally, preventing "LLM Math Hallucinations."
*   **Smart Timeframes:** Automatically downloads 1-Hour data and resamples it into **4-Hour Candles** for Swing Trading analysis using Pandas.
*   **Recency Filters:** Enforces strict time limits (`timelimit='d'`) on search results to ensure the AI reacts to *today's* news, not last year's.
*   **Date Awareness:** The system injects the current date into the Agent's context, allowing it to function correctly in any timeframe.

## üß† System Architecture

The system follows a sequential process:

1.  **Manager** searches DuckDuckGo for keywords like "Fed Outlook" and "Gold Price News."
2.  **Manager** verifies the dates of the articles and outputs a Sentiment Report (Bullish/Bearish/Neutral).
3.  **Analyst** receives this report via context injection.
4.  **Analyst** pulls the latest 4H candles, RSI, and EMA data.
5.  **Analyst** determines if the Technicals match the Fundamentals.
6.  **Final Output:** A structured Trade Signal (Entry, Stop Loss, Take Profit).
